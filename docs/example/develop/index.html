    CatMq的领域模型 #

    CatMq 是一款在kafka基础上发展出来的一款典型的消息队列中间件，使用异步通信的方式和发布订阅模型消息传输队列。通信方式和传输模型的具体说明可以看下文的通信方式介绍和消息传输模型。CatMq具有异步通信的优势，系统拓扑简单，系统间耦合低，主要应用于异步解耦，流量削峰填谷的场景。
